# 웹사이트 스크린샷 생성 가이드

## 설치 방법

1. **Node.js 설치 확인**
   ```bash
   node --version
   ```
   Node.js가 설치되어 있지 않다면 [nodejs.org](https://nodejs.org/)에서 다운로드하세요.

2. **의존성 설치**
   ```bash
   npm install
   ```
   이 명령어는 `puppeteer`를 설치합니다 (Chrome 브라우저 포함).

## 사용 방법

### 모든 페이지 스크린샷 생성
```bash
npm run screenshot
```

또는

```bash
node screenshot.js
```

## 생성되는 파일

각 페이지마다 3가지 스크린샷이 생성됩니다:

1. **`{페이지명}-full.png`**: 전체 페이지 스크린샷 (스크롤 포함)
2. **`{페이지명}-viewport.png`**: 첫 화면만 스크린샷 (1920x1080)
3. **`{페이지명}-mobile.png`**: 모바일 뷰 (375x667)

## 생성되는 스크린샷 목록

- `index-full.png`, `index-viewport.png`, `index-mobile.png`
- `plan-full.png`, `plan-viewport.png`, `plan-mobile.png`
- `schedule-full.png`, `schedule-viewport.png`, `schedule-mobile.png`
- `my-trips-full.png`, `my-trips-viewport.png`, `my-trips-mobile.png`
- `calendar-full.png`, `calendar-viewport.png`, `calendar-mobile.png`

## 저장 위치

모든 스크린샷은 `screenshots/` 폴더에 저장됩니다.

## 문제 해결

### Puppeteer 설치 오류
- 인터넷 연결을 확인하세요 (Chrome 브라우저 다운로드 필요)
- 관리자 권한으로 실행해보세요

### 스크린샷이 비어있거나 깨짐
- HTML 파일 경로를 확인하세요
- `styles.css`와 다른 리소스 파일들이 같은 폴더에 있는지 확인하세요

### 지도가 로드되지 않음
- `schedule.html`의 경우 지도 로딩 시간이 필요합니다
- 스크립트에서 자동으로 3초 대기합니다

## 수동 스크린샷 방법 (대안)

스크립트가 작동하지 않는 경우:

1. **브라우저 개발자 도구 사용**
   - F12 → Console 탭
   - 다음 코드 실행:
   ```javascript
   // 전체 페이지 스크린샷 (Chrome 확장 프로그램 필요)
   // 또는 브라우저 확장 프로그램 사용:
   // - "Full Page Screen Capture" (Chrome)
   // - "FireShot" (Chrome/Firefox)
   ```

2. **온라인 도구 사용**
   - [htmlcsstoimage.com](https://htmlcsstoimage.com/)
   - [urlbox.io](https://www.urlbox.io/)

3. **Windows 기본 도구**
   - `Win + Shift + S`: 스크린샷 도구
   - 브라우저에서 페이지 열고 캡처
