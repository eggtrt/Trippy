# TRIPPY 기획서

## 1. 서비스 개요

### 1.1 서비스명
**TRIPPY** - 나만을 위한 AI 기반 개인화 여행 계획 서비스

### 1.2 서비스 소개
AI가 사용자의 취향을 학습하고, 여행 계획을 자동으로 생성하여 Google Calendar에 동기화해주는 스마트 여행 계획 플랫폼입니다.

### 1.3 핵심 가치 제안
- **AI 기반 자동 계획 생성**: 복잡한 여행 계획을 AI가 자동으로 생성
- **Google Calendar 자동 동기화**: 생성된 일정이 자동으로 캘린더에 반영
- **지도와 일정 통합 뷰**: 지도에서 여행 경로를 시각적으로 확인
- **개인화된 여행 경험**: 사용자의 취향과 스타일에 맞춘 맞춤형 계획

---

## 2. 핵심 기능

### 2.1 AI 자동 계획 생성
- **기능 설명**: AI 챗봇과의 자연스러운 대화를 통해 여행 계획 자동 생성
- **주요 특징**:
  - 단계별 질문을 통한 사용자 취향 파악
  - 도시, 기간, 동행자, 여행 스타일 기반 맞춤 계획
  - 일정과 동선을 고려한 최적 시간 배치
- **사용자 경험**:
  - 간단한 UI를 통한 정보 입력
  - AI 챗봇을 통한 추가 요청사항 반영
  - 실시간 계획 생성 및 미리보기

### 2.2 Google Calendar 자동 동기화
- **기능 설명**: 생성된 여행 계획이 Google Calendar에 자동으로 추가
- **주요 특징**:
  - Google 로그인으로 간편한 연동
  - 일정 자동 동기화
  - 동선을 고려한 시간 배치
  - 카테고리별 색상 구분 (식사, 관광, 이동, 숙소)
- **사용자 경험**:
  - 원클릭 동기화
  - 캘린더에서 바로 확인 가능
  - 수정 및 업데이트 반영

### 2.3 지도와 일정 통합 뷰
- **기능 설명**: 지도에서 여행 경로를 시각적으로 확인하고 일정과 함께 관리
- **주요 특징**:
  - 지도에서 경로 시각화
  - 일정과 위치 동시 확인
  - 최적 이동 경로 제안
  - 카테고리별 마커 표시
- **사용자 경험**:
  - 지도와 일정 리스트 동시 표시
  - 일정 클릭 시 지도에서 위치 확인
  - 경로 자동 연결 및 시각화

### 2.4 여행 계획 관리
- **기능 설명**: 생성된 여행 계획을 저장하고 관리
- **주요 특징**:
  - 내 여행 목록 보기
  - 저장된 여행 일정 확인
  - 캘린더 뷰에서 전체 일정 확인
  - 일정 수정 및 업데이트

## 3. 사용자 여정 (User Journey)

### 3.1 여행 계획 생성 프로세스

#### Step 1: Google 로그인
- Google 계정으로 간편 로그인
- Calendar 연동 설정

#### Step 2: 여행 정보 입력
**2-1. 도시 선택**
- 일본: 도쿄, 오사카, 후쿠오카, 오키나와, 삿포로
- 대한민국: 제주, 부산, 경주, 강릉·속초, 전주
- 중화/중국: 홍콩, 타이베이, 상하이, 베이징
- 동남아시아: 방콕, 다낭, 푸켓, 싱가포르
- 유럽: 파리, 로마, 런던, 바르셀로나

**2-2. 기간 선택**
- 당일치기, 1박 2일, 2박 3일, 3박 4일, 4박 5일, 5박 6일

**2-3. 동행자 선택**
- 혼자, 연인, 친구, 가족, 동료

**2-4. 여행 스타일 선택** (다중 선택 가능)
- 체험·액티비티
- SNS 핫플레이스
- 자연과 함께
- 유명 관광지는 필수
- 여유롭게 힐링
- 문화·예술·역사
- 여행지 느낌 물씬
- 쇼핑은 열정적으로
- 관광보다 먹방

#### Step 3: AI 챗봇 상호작용
- 입력한 정보 요약 확인
- 추가 요청사항 입력 (예: "맛집 위주로", "힐링 중심으로")
- AI가 맞춤 계획 생성

#### Step 4: 계획 확인 및 저장
- 생성된 일정 확인
- 지도에서 경로 확인
- 일정 수정 및 조정
- 저장 및 Calendar 동기화

### 3.2 일정 관리 프로세스
1. **내 여행 보기**: 저장된 여행 목록 확인
2. **일정 상세 보기**: 지도와 일정 통합 뷰에서 확인
3. **캘린더 확인**: 주간/월간 캘린더 뷰에서 전체 일정 확인
4. **일정 수정**: 필요시 일정 수정 및 업데이트

## 4. 기술 스택

### 4.1 Frontend
- **HTML5/CSS3**: 반응형 웹 디자인
- **JavaScript**: 인터랙티브 UI 및 로직 구현
- **Leaflet.js**: 지도 표시 및 경로 시각화 (OpenStreetMap)
- **Google APIs**: 
  - Google Sign-In (OAuth)
  - Google Calendar API (동기화)

### 4.2 주요 기능 구현
- **로컬 스토리지**: 여행 일정 임시 저장
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 지원
- **캘린더 UI**: Google Calendar 스타일의 주간/월간 뷰

## 10. 결론

TRIPPY는 AI 기술과 Google Calendar 연동을 통해 사용자가 복잡한 여행 계획을 간편하게 생성하고 관리할 수 있도록 돕는 혁신적인 여행 계획 플랫폼입니다. 

MVP 단계에서 핵심 기능을 구현하여 사용자 피드백을 받고, 지속적인 개선을 통해 완성도 높은 서비스로 발전시켜 나갈 계획입니다.
