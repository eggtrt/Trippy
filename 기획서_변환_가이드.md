# TRIPPY 기획서 변환 가이드

## PDF로 변환하는 방법

### 방법 1: 온라인 도구 사용
1. **Markdown to PDF** (https://www.markdowntopdf.com/)
   - `TRIPPY_기획서.md` 파일 업로드
   - PDF 다운로드

2. **Pandoc 사용** (명령줄)
   ```bash
   pandoc TRIPPY_기획서.md -o TRIPPY_기획서.pdf
   ```

### 방법 2: VS Code 확장 프로그램
1. VS Code에서 "Markdown PDF" 확장 설치
2. `TRIPPY_기획서.md` 파일 열기
3. `Ctrl+Shift+P` → "Markdown PDF: Export (pdf)" 선택

### 방법 3: Google Docs 사용
1. Google Docs 열기
2. `TRIPPY_기획서.md` 내용 복사/붙여넣기
3. 서식 조정 후 파일 → 다운로드 → PDF

## PPT로 변환하는 방법

### 방법 1: Google Slides 사용 (추천)
1. Google Slides 열기
2. 각 섹션을 슬라이드로 분리:
   - 슬라이드 1: 표지 (서비스명, 로고)
   - 슬라이드 2: 서비스 개요
   - 슬라이드 3: 핵심 기능 (3개)
   - 슬라이드 4: 사용자 여정
   - 슬라이드 5: 기술 스택
   - 슬라이드 6: UI/UX 디자인
   - 슬라이드 7: MVP 범위
   - 슬라이드 8: 비즈니스 모델
   - 슬라이드 9: 경쟁 우위
   - 슬라이드 10: 개발 로드맵
   - 슬라이드 11: 결론

3. 웹사이트 스크린샷 추가
4. 다운로드 → PowerPoint 형식

### 방법 2: PowerPoint 직접 작성
1. PowerPoint 열기
2. 마크다운 내용을 슬라이드별로 정리
3. 웹사이트 스크린샷 삽입
4. 차트 및 다이어그램 추가

### 방법 3: 온라인 도구
- **Slides.com**: 마크다운을 슬라이드로 변환
